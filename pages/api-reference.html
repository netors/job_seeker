API Reference

Complete command reference and configuration options for the AI-powered job search system

Command Line Interface

Basic Commands

RUN CrewAI Job Search Workflow

crewai run

Executes the complete AI-powered job search workflow using CrewAI's multi-agent orchestration. Watch each agent's reasoning and decision-making process in real-time.

Prerequisites:
  • Configured user profile in knowledge/resume_template.json
  • Valid API keys in .env file
  • CrewAI project installed with crewai install
AI Workflow Output:
  • Real-time display of agent reasoning and tool usage
  • job_search_report.md - Comprehensive analysis and recommendations
  • application_strategy.md - Personalized application guidance
  • job_opportunities.db - Complete SQLite database
  • Use crewai run > output.log to save the complete workflow

TRAIN Train AI Agents

crewai train

Trains the AI agents to improve their performance through iterative learning and optimization.

Training Benefits:
  • Improves agent decision-making accuracy
  • Optimizes tool usage and workflow efficiency
  • Enhances job matching algorithm performance
  • Reduces execution time through learned optimizations

TEST Test System

crewai test

Runs comprehensive tests to verify all agents, tools, and system components are functioning correctly.

Test Coverage:
  • Agent initialization and configuration
  • Tool functionality and API connectivity
  • Database operations and schema validation
  • File system permissions and output generation
  • End-to-end workflow execution

REPLAY Replay Tasks

crewai replay <task_id>

Replays specific workflow tasks for debugging, analysis, or performance optimization.

task_id string required

Unique identifier of the task to replay (found in workflow logs or training output).

Use Cases:
  • Debug specific agent failures or errors
  • Analyze decision-making patterns
  • Test workflow modifications
  • Performance profiling and optimization

Advanced CrewAI Commands

INSTALL Install Dependencies

crewai install

Installs all required dependencies and sets up the CrewAI project structure for optimal performance.

Installation Features:
  • Installs CrewAI framework and all dependencies
  • Sets up virtual environment if needed
  • Configures project structure and paths
  • Validates Python version compatibility
  • Prepares tools and agent configurations

OUTPUT Save Workflow Output

crewai run > workflow_output.log

Captures the complete AI agent workflow, including reasoning, tool usage, and decision-making processes.

Captured Information:
  • Agent thought processes and reasoning chains
  • Tool execution details and API responses
  • Inter-agent communication and data handoffs
  • Performance metrics and execution timing
  • Error handling and recovery mechanisms

ANALYZE Workflow Analysis

crewai replay <task_id> --analyze

Provides detailed analysis of specific workflow executions for optimization and debugging.

Analysis Features:
  • Performance bottleneck identification
  • Agent decision-making pattern analysis
  • Tool usage efficiency metrics
  • Error pattern recognition
  • Optimization recommendations

Configuration Reference

Environment Variables (.env)

Required Variables

SERPER_API_KEY string required

SerperDev API key for web search functionality. Get free tier at serper.dev.

Optional Variables

OPENAI_API_KEY string optional

OpenAI API key for enhanced AI capabilities. Recommended for best performance.

MODEL string optional

AI model to use. Default: gpt-4o. Options: gpt-3.5-turbo, gpt-4, gpt-4o.

CREWAI_DISABLE_TELEMETRY boolean optional

Disable CrewAI telemetry collection. Default: false. Set to true for privacy.

CREWAI_DISABLE_TRACKING boolean optional

Disable CrewAI usage tracking. Default: false. Set to true for privacy.

OTEL_SDK_DISABLED boolean optional

Disable OpenTelemetry SDK. Default: false. Set to true to disable tracing.

User Profile Schema (knowledge/resume_template.json)

Personal Information

name string required

Your full name as it should appear in job applications.

email string required

Primary email address for job applications.

phone string optional

Phone number including country/area code.

Professional Information

current_role string required

Your current job title or target role if unemployed.

years_experience number required

Total years of professional experience in your field.

skills array[string] required

List of technical skills, programming languages, frameworks, and tools.

location string required

Your current location (city, state/province, country).

preferred_locations array[string] required

List of locations where you're willing to work. Include "Remote" if applicable.

expected_salary number required

Expected annual salary in your local currency (base salary, not total compensation).

preferred_company_type string optional

Preferred company type. Options: startup, enterprise, nonprofit, healthcare_tech, fintech.

Education and Experience

Educational background with degree, institution, and graduation year.

experience array[object] optional

Work experience with titles, companies, duration, and achievements.

certifications array[string] optional

Professional certifications and their years of completion.

projects array[object] optional

Notable projects with descriptions, technologies used, and URLs.

languages array[string] optional

Spoken languages with proficiency levels.

interests array[string] optional

Professional interests and areas of expertise.

Custom Search Parameters

search_custom JSON Schema

job_sites array[string] optional

Specific job sites to search. Options: indeed.com, linkedin.com, glassdoor.com, dice.com.

search_query string optional

Custom search query to override profile-based search terms.

location string optional

Specific location for this search, overriding profile preferences.

max_results number optional

Maximum number of results to return. Default: 20. Range: 10-100.

salary_min number optional

Minimum salary filter for search results.

experience_level string optional

Target experience level. Options: entry, mid, senior, lead, executive.

Complete Example: 
{
  "job_sites": ["indeed.com", "linkedin.com"],
  "search_query": "Senior Full Stack Developer",
  "location": "San Francisco Bay Area",
  "max_results": 50,
  "salary_min": 120000,
  "experience_level": "senior"
}

Output Formats

Generated Files

job_search_report.md

Comprehensive markdown report with the following sections:

Report Structure:
  • Executive Summary: Overview of search results and key findings
  • Top Opportunities: Ranked list of best matches with scores
  • Match Analysis: Detailed scoring breakdown for each recommendation
  • Skills Gap Analysis: Identified missing skills and improvement areas
  • Market Insights: Salary trends and demand analysis
  • Company Analysis: Information about recommended companies
  • Search Statistics: Metrics about the search execution

application_strategy.md

Personalized application guidance document containing:

Strategy Components:
  • Application Priority: Recommended order for applying to opportunities
  • Cover Letter Templates: Customized templates for top opportunities
  • Interview Preparation: Common questions and talking points
  • Company Research: Key insights about target companies
  • Timeline Recommendations: Suggested application schedule
  • Networking Strategies: Contact recommendations and approaches
  • Skill Development: Priority areas for improvement

job_opportunities.db

SQLite database with comprehensive job data storage:

Database Tables:
  • job_opportunities: Complete job listings with match scores
  • search_history: Record of all search executions
  • evaluation_criteria: Scoring algorithm details
  • application_tracking: Status of job applications (future use)
Query Examples: 
-- Top 10 opportunities by match score
SELECT title, company, location, match_score
FROM job_opportunities
ORDER BY match_score DESC
LIMIT 10;

-- Jobs by company type
SELECT company, COUNT(*) as job_count
FROM job_opportunities
GROUP BY company;

-- Search history summary
SELECT search_date, results_count, execution_time
FROM search_history
ORDER BY search_date DESC;

Error Codes and Troubleshooting

Common Error Messages

Configuration Errors

CrewAI Command Purpose Description
crewai run Execute AI Workflow Run the complete job search with AI agent collaboration
crewai train Improve Performance Train agents for better decision-making and efficiency
crewai test Validate System Test all components, agents, and integrations
crewai replay Debug & Analyze Replay specific tasks for debugging and optimization
crewai install Setup Project Install dependencies and configure CrewAI project
Error Message Cause Solution
No user profile found Missing or invalid profile file Create/fix knowledge/resume_template.json
Invalid JSON in profile Syntax error in profile file Validate JSON syntax and fix errors
API key not found Missing environment variables Set SERPER_API_KEY in .env file
Invalid API key format Malformed API key Check API key format and regenerate if needed

Runtime Errors

Error Message Cause Solution
SerperDev API timeout Network or API service issues System falls back to mock data automatically
Database connection failed File permissions or disk space Check write permissions in project directory
Module not found Missing dependencies Run pip install -e . again
Agent initialization failed CrewAI configuration issue Check agent and task YAML files

Exit Codes

Exit Code Meaning Description
0 Success Command completed successfully
1 General Error Unspecified error occurred
2 Configuration Error Invalid configuration or missing files
3 API Error External API failure or authentication
4 Database Error Database operation failed
5 Agent Error AI agent execution failed
