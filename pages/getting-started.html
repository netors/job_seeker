Getting Started

Complete setup guide to get your AI-powered job search system running in minutes

Prerequisites

üêç Python Environment

  • Python 3.10 or higher (supports up to 3.13)
  • Virtual environment support (venv or conda)
  • pip package manager
  • Git for version control
python --version  # Should be 3.10+

üîë API Keys Required

  • SerperDev API Key (Required - Free tier available)
  • OpenAI API Key (Recommended for best performance)

Don't worry, we'll walk through getting these in the setup process.

Installation Steps

Step 1: Clone and Install

First, clone the repository and install the package in development mode:

# Clone the repository
git clone https://github.com/netors/job_seeker.git
cd job_seeker

# Create and activate virtual environment (recommended)
python -m venv .venv
source .venv/bin/activate  # On Windows: .venv\Scripts\activate

# Install CrewAI project dependencies
crewai install

What this does:

  • Downloads the complete source code to your machine
  • Creates an isolated Python environment for the project
  • Installs all required CrewAI dependencies automatically
  • Sets up the CrewAI project structure and commands

Step 2: Environment Configuration

Set up your environment variables for API keys and configuration:

# Copy the environment template
cp env_template.txt .env

# Edit the .env file with your API keys
nano .env  # or use your preferred editor

Your .env file should look like this:

# Required: SerperDev API for job search
SERPER_API_KEY=your_serper_api_key_here

# Recommended: OpenAI API for best AI performance
OPENAI_API_KEY=your_openai_api_key_here

# Model selection (optional)
MODEL=gpt-4o

# Disable telemetry (optional)
CREWAI_DISABLE_TELEMETRY=true
CREWAI_DISABLE_TRACKING=true
OTEL_SDK_DISABLED=true

Step 3: Get Your API Keys

üîç SerperDev API Key (Required)

  1. Visit serper.dev
  2. Sign up for a free account
  3. Get 100 free searches per month
  4. Copy your API key to the .env file

Why needed: This powers the job search across multiple platforms.

ü§ñ OpenAI API Key (Recommended)

  1. Visit platform.openai.com
  2. Create an account or sign in
  3. Generate an API key
  4. Add it to your .env file

Why recommended: Provides the best AI performance for job evaluation and strategy generation.

Profile Configuration Tips

üí° Skills Section

  • Be specific: Include both technical skills and frameworks
  • Include levels: Mention proficiency if relevant
  • Add soft skills: Leadership, communication, problem-solving
  • Stay current: Focus on technologies you're actively using

üìç Location Preferences

  • Be flexible: Include multiple cities you'd consider
  • Remote work: Add "Remote" if you're open to it
  • Commute radius: Include nearby cities within commuting distance
  • Relocation: Add cities you'd relocate for the right opportunity

üí∞ Salary Expectations

  • Market research: Check salary ranges for your role and location
  • Be realistic: Based on your experience level
  • Total compensation: Consider base salary, not total comp
  • Growth trajectory: Factor in career progression goals

üè¢ Company Preferences

  • Company type: startup, enterprise, nonprofit, healthcare_tech
  • Industry focus: Consider what industries interest you
  • Company size: Small teams vs large organizations
  • Culture fit: Fast-paced vs stable environments

Troubleshooting

Common Issues and Solutions

‚ùå "No user profile found"

Problem: The system can't find your profile configuration.

Solution:

  1. Check that knowledge/resume_template.json exists
  2. Verify the JSON syntax is valid
  3. Ensure all required fields are present
  4. Ensure your profile in knowledge/resume_template.json is properly configured

üîë "API key not found"

Problem: Missing or incorrect API key configuration.

Solution:

  1. Check that .env file exists in the project root
  2. Verify your API keys are correctly set
  3. Ensure no extra spaces around the = sign
  4. Restart your terminal after setting environment variables

üåê "SerperDev API timeout"

Problem: Network issues or API service problems.

Solution:

  1. Check your internet connection
  2. Verify your SerperDev API key is valid
  3. The system will automatically fall back to mock data
  4. Try running the search again in a few minutes

üì¶ "Module not found"

Problem: Dependencies not properly installed.

Solution:

  1. Ensure you're in the correct directory
  2. Activate your virtual environment
  3. Run crewai install again
  4. Check Python version is 3.10 or higher
